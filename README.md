# Resident Evil 4 Inventory Portfolio

A Next.js 16 + Tailwind CSS portfolio inspired by the Resident Evil 4 attaché case inventory screen.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run lint` runs Next's lint command.

Portfolio content lives in `data/inventory.ts`. Update that file to add projects, technologies, work history, social links, or contact details.
